🌤️ Weather Data Pipeline System

A complete End-to-End ETL Pipeline that extracts real-time weather data for major Indian cities, transforms it, loads it into SQLite database, and generates automated reports with alerts.

✨ Features
Real-time weather data collection for 6 Indian cities (Mumbai, Delhi, Bangalore, Chennai, Kolkata, Hyderabad)

SQLite database with normalized tables (cities & weather_data)

ETL Pipeline with proper error handling and logging
Temperature & Humidity based alert system

Clean, formatted reporting with latest data only

No API key required (using Open-Meteo + fallback mechanism)

Easy to run and extend

🛠️ Tech Stack

Python 3

SQLite (Database)

Requests (API calls)

Open-Meteo API (Free weather data)
